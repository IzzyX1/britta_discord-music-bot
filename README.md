# Britta discord-music-bot

## installation
- git clone https://github.com/b3h3m0th/britta_discord-music-bot.git
- cd britta_discord-music-bot
- install dependencies (npm install)
- invite to your discord server


## commands

### prefix: +

- +play
- +join
- +leave
- +stop
- +resume
- +setVolume
- +help
- +clear
- +queue
- +britta
- +gibarua
